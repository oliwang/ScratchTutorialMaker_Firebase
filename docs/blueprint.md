# **App Name**: Scratch Guide

## Core Features:

- Project Import: Import Scratch project from a local .sb3 file or a scratch.mit.edu URL.
- Tutorial Generation: Analyze the Scratch project and generate a step-by-step tutorial, including a general description, resource extraction, and function-based instructions suitable for young learners. The LLM will use the project information as a tool.
- Google Docs Export: Export the generated tutorial to a Google Docs document.

## Style Guidelines:

- Primary color: Use a muted teal (#607c8a) for headers and main elements.
- Secondary color: Use a light beige (#f5f5dc) for backgrounds to ensure readability.
- Accent: Use a dusty rose (#bc8f8f) for buttons and interactive elements to draw attention without being too loud.
- Maintain clear and legible typography throughout the app.
- Employ simple, intuitive icons to represent actions and resources.
- Use a clean, organized layout to present the tutorial steps clearly.
- Incorporate subtle animations for transitions and user feedback.
